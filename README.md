Aplicación web desarrollada con React + TypeScript para gestionar hábitos y tareas diarias.

# Tecnologías utilizadas
- React
- TypeScript
- Vite
- CSS
- LocalStorage

# Funcionalidades

- Crear hábitos
- Marcar hábitos como completados
- Eliminar hábitos
- Persistencia de datos con LocalStorage
- Filtrado de hábitos
- Manejo de estado con useReducer

  # Este proyecto fue realizado como práctica personal para seguir aprendiendo frontend moderno con React y TypeScript.

Principalmente trabajé:
- tipado con TypeScript
- manejo de estado global
- reducers y dispatch
- persistencia de datos
- organización de componentes
